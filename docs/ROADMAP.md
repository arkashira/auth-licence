# ROADMAP.md

## Roadmap - auth-licence

### MVP (Minimum Viable Product) - Q3 2026

**Theme: Core Authentication & Licensing Foundation**

**MVP-Critical Features:**
- **Basic Authentication System**
  - Email/password authentication with secure hashing
  - JWT token generation and validation
  - User registration and login flows
  - Password reset functionality

- **Simple Licensing Module**
  - License key generation and validation
  - Basic license tiers (Free, Basic, Pro)
  - License expiration handling
  - Offline license validation capability

- **Developer Integration SDK**
  - SDK for Node.js, Python, and PHP
  - Simple API endpoints for integration
  - Basic documentation and examples
  - Quick-start guide for developers

- **Dashboard for Developers**
  - License key management
  - Basic usage analytics
  - User management overview
  - Simple fraud detection alerts

- **Security Foundation**
  - HTTPS enforcement
  - Rate limiting on authentication endpoints
  - Basic protection against brute force attacks
  - Secure data storage practices
